ORDER OF FILES

1. 'dbconfig.PY' CREATES THE 'mydb_config.ini' THAT CONTANS THE CONFIGURATION FILES TO ACCESS THE DATABASE.
2. 'datainjection_todb.py' RUNS THE 'createdb.sh' FILE TO CREATE TEH DATABASE. IT THEN LOADS THE DATA FROM THE CSV FILE USING THE 'getdata_from_csv.py' FILE AND USES THE DATA TO CRETAE A TABLE IN THE DATABSE WITH THE 'dbconfig.ini' FILE.
3. 'loaddata_fromdb.py' CONTAINS A FUNCTION THAT TAKES THE NAME OF THE TABLE IN THE DATABSE AND RETURNS THE DATA IN IT AS A DATAFRME.
